Welcome

This website is intended to teach the bare basics of music theory. It teaches just enough to allow you to create music and interact with other musicians. This is good for beginners or experienced musicians, who need to fill some knowledge gaps. To help explain things I have used standard notation. I only use very basic written music, and I explain how to read it, so please don't be threatened by this. These are the basic building blocks that artists use to express themselves and convey their experiences. In learning these things I hope you can have fun doing this too. Please click the titles across to explore the different areas. These are easier to understand if you read them in the order presented, although this is not essential.
